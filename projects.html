MINDIT - Mental Well-being Application

Designed to assist doctors with patient data management and early mental health screening. It's built using Flutter for UI and Firebase for data storage and multilingual support.

MINDIT is a mobile solution deployed at Government Rajaji Hospital, Madurai, for screening and categorizing mental well-being among patients. It uses a rule-based decision tree model to analyze mental health severity levels and is particularly useful in underserved communities.

The app supports multiple languages and provides an intuitive interface for self or assisted mental wellness checks. It includes early-intervention features for at-risk individuals and assists doctors with categorized screening data.

Tech Stack: Flutter, Firebase (Firestore & Auth), Dart, Decision Tree Algorithm

Outcome: Enabled early mental health screening and categorization for patients
